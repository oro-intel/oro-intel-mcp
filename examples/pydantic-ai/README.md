# PydanticAI

A PydanticAI agent using Oro Intel's tools over the **remote MCP server** (`MCPServerStreamableHTTP`) to resolve a company and fetch its full profile, reporting credits used.
Run: `pip install "pydantic-ai-slim[mcp,anthropic]"` then `ORO_API_KEY=oro_... ANTHROPIC_API_KEY=... python agent.py`.
