# LlamaIndex

A LlamaIndex FunctionAgent using Oro Intel's tools over the **remote MCP server** (`llama-index-tools-mcp`) to resolve a company and fetch its full profile, reporting credits used.
Run: `pip install llama-index llama-index-tools-mcp llama-index-llms-anthropic` then `ORO_API_KEY=oro_... ANTHROPIC_API_KEY=... python agent.py`.
